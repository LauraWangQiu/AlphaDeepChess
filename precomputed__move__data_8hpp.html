typedef std::array<std::array<uint64_t, BISHOP_TABLE_SIZE>, NUM_SQUARES> TableBishopMoves

Lookup table for bishop moves.

This typedef represents a 2D array where each element corresponds to a precomputed bitboard of possible bishop moves. The first dimension is indexed by the square where the bishop is located, and the second dimension is indexed by the blocker configuration.
