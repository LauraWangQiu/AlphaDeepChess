Detailed Description

TranspositionTable.

Transposition table used as cache to store evaluation of chess positions

Member Enumeration Documentation

◆ NodeType

enum class TranspositionTable::NodeType : uint8_t
strong

TranspositionTable::NodeType.

We usually do not find the exact value of a position but we can know that the value is either too low or too high for us to be concerned with searching any further. If we have the exact value, of course, we store that in the transposition table. But if the value of our position is either high enough to set the lower bound, or low enough to set the upper bound, it is good to store that information also.
