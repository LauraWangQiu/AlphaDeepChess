Detailed Description

Move.

Represents a chess move. Move::null() = 0.

Move is valid if origin_square != destination_square.

Note
Don't modify the bitfields they are used in other functions. move is stored as a 16-bit number:

14-15: special move flag: NORMAL(0), PROMOTION(1), EN_PASSANT(2), CASTLING(3). 12-13: promotion piece : KNIGHT(0), BISHOP(1), ROOK(2), QUEEN(3). 6-11: square from (origin) : (0 to 63). 0- 5: square to (destination): (0 to 63).
